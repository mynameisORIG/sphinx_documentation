Find a User’s SID in Windows

Find a User’s SID with WMIC

All Users

Open up a command prompt and type the following:

wmic useracount get name,sid

Single User

wmic useraccount where name="USER" get sid

Finding a Username Using the SID

wmic useraccount where sid="S-I-D-Number" get name

Find a User;s SID in the REGISTRY

Go into this registry path HKEY_LOCAL_MACHINESOFTWAREMicrosoftWindows NTCurrentVersionProfileList and look for ProfileImagePath
