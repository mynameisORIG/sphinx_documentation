Linux boot issues

SCSI Disk (0,0): The signed image’s hash is not allowed (DB) fix

  1. Open Hyper-V Manager.

  2. Select the VM showing the error.

  3. Right-click the VM and select Settings.

  4. In the Settings window, navigate to Security.

  5. Uncheck “Enable Secure Boot” and click OK.

  6. Reboot the VM to see if it boots properly.
