Feroxbuster web enumeration
******************************

enum web pages
############################

The `-u` stands for url. The `-w` stands for wordlist. The `-s` stands for html code. 200 stands for running web page. The `-x` stands for what type of web page

.. code-block:: console

   feroxbuster -u http://<ip>:<optional port> -w <wordlist.txt> -s 200 -d 1 -x php
