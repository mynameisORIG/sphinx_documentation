general information
***************************

console.log
####################

This outputs stuff to the terminal

.. code-block:: console

    console.log("Hello World")

export
###############

This turns the typescript file into a module. This is declared at the end of the file.

.. code-block:: console

    export {}

Compile and run
####################

.. code-block:: console

    tsc file.ts
    node file.js
