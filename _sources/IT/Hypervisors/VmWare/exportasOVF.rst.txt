export as OVF
***********************

.. _link2: https://masteringvmware.com/how-to-export-vm-as-ovf-template-in-vcenter-server-7/

#. Open vSphere Client
#. Find the VM you're interested in
#. Right-Click the VM -> Export -> Export OVF Template
  #. In the OVF template dialog, type the template name (ex. NewVM)
  #. Click Ok
